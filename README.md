# Telfer B.Com (1st Year) Reinvention — Interactive Web Brochure

This is a static, GitHub Pages–ready brochure (HTML/CSS/JS) that contains the full report content + quote callouts.

## Quick start (GitHub Pages)
1. Upload **index.html**, **styles.css**, **script.js**, and the **assets** folder to the root of your repo.
2. In GitHub: **Settings → Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` / root
3. Wait for the URL GitHub provides.

## Logos
Replace the placeholder images in `/assets`:
- `logo-telfer.png`
- `logo-green-academy.png`
- `logo-prefab.png`

## Optional: add audio later
Create `/audio` and add:
- `prof_1.mp3`, `stu_1.mp3` … `prof_10.mp3`, `stu_10.mp3`

Then we can add an interview player UI.

## Notes
- Use **Print / Save PDF** to export a clean PDF version.
- Use the sidebar search to quickly find topics (e.g., AI, Brightspace, feedback).
